|Term | Переклад |Meaning|
|-----|----------|-------|
|Router|Маршрутизатор|A network device that connects different networks (e.g., LANs and WANs) and makes routing decisions to forward packets based on the destination IP address|
|Switch|Комутатор|A network device that connects end devices within the same Local Area Network (LAN). It forwards frames based on the destination MAC address|
|Subnet Mask|Маска підмережі|A 32-bit number used with the IP Address to determine which part of the address belongs to the network and which part belongs to the host (device)|
|Default Gateway|Шлюз за замовчуванням|The IP Address of the router (or other device) to which an end device sends all packets destined for other networks|
|ARP (Address Resolution Protocol)|Протокол визначення адрес|A protocol used to map an IP address (Layer 3) to its corresponding MAC address (Layer 2) within a local network segment|
|DHCP (Dynamic Host Configuration Protocol)|Протокол динамічного налаштування хостів|A protocol that automatically assigns IP addresses, subnet masks, default gateways, and other network parameters to end devices.|
|Console Cable|Консольний кабель|A cable (e.g., RS-232 or USB) used for direct connection from a computer to the console port of a router or switch for initial configuration.|
|VLAN (Virtual Local Area Network)|Віртуальна локальна мережа|A method of logically segmenting a single physical LAN into multiple smaller, independent networks on a switch|
|CLI (Command Line Interface)|Інтерфейс командного рядкаA text-based interface used to configure Cisco routers and switches by typing in commands|
|Traceroute/Tracert|Трасування маршруту|A diagnostic utility that displays the route and transit times of packets to a destination, showing all routers (hops) along the path|
